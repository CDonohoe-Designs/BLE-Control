# Altium AD25 – PCB Shortcuts (Quick Reference)

> Focused on placement, routing and inspection – the ones you’ll use all the time.

---

## 1. Selection & Nets

| Action                              | Shortcut                            |
|-------------------------------------|-------------------------------------|
| Select whole net                    | Click a track/pad, then **`Tab`**   |
| Grow selection one level            | **`Tab`** again                     |
| Clear selection                     | **`Shift` + `C`**                   |
| Toggle between mm ↔ mil             | **`Q`**                             |

---

## 2. View / Grids / Layers

| Action                              | Shortcut                                            |
|-------------------------------------|-----------------------------------------------------|
| Zoom in/out                         | Mouse wheel                                         |
| Pan                                 | Click & hold **right mouse** + drag                |
| Fit board to window                 | **`V`**, then **`B`** (`View → Fit Board`)          |
| Fit all objects (document)          | **`V`**, then **`D`**                               |
| Toggle grids on/off                 | **`G`**                                             |
| Grid settings dialog                | **`Shift` + `G`**                                   |
| Single-layer mode (cycle layers)    | **`Shift` + `S`** (tap repeatedly)                  |
| View configuration (layers/colors)  | **`L`**                                             |

---

## 3. Moving & Placing Components

_(Start by hovering the component, then press the shortcut.)_

| Action                              | Shortcut                            |
|-------------------------------------|-------------------------------------|
| Move component                      | **`M`**, then **`C`**               |
| Move selection                      | **`M`**, then **`M`**               |
| Copy selection                      | **`C`**, then **`C`**               |
| Rotate while moving/placing         | **`Space`**                         |
| Flip horizontally while moving      | **`X`**                             |
| Flip vertically while moving        | **`Y`**                             |

---

## 4. Interactive Routing

| Action                              | Shortcut                                                |
|-------------------------------------|---------------------------------------------------------|
| Start interactive routing           | **`Ctrl` + `W`**                                        |
| Change corner style / walkaround    | **`Shift` + `Space`** (cycle modes)                     |
| Toggle route direction at corner    | **`Space`** while routing                              |
| Change active layer & drop via      | **`*` (numeric keypad)** while routing                  |
| Finish current route                | **Right-click** or **`Ctrl` + click** at end            |
| Change current track width          | **`Shift` + `W`**                                       |

---

## 5. Vias & Polygons

| Action                              | Shortcut                            |
|-------------------------------------|-------------------------------------|
| Place via (while not routing)       | **P**, then **V** (`Place → Via`)   |
| Toggle single-layer (poly debug)    | **`Shift` + `S`**                   |
| Repour all polygons                 | **`T`**, then **`G`**, then **`A`** |

---

## 6. DRC & Reports

| Action                              | Shortcut                                    |
|-------------------------------------|---------------------------------------------|
| Run Design Rule Check (DRC)         | **T**, then **D**, then **R**               |
| Open Messages panel (DRC results)   | **`View → Panels → Messages`** (no default) |

---

## 7. Handy Schematic Shortcuts (for context)

| Action                              | Shortcut                            |
|-------------------------------------|-------------------------------------|
| Compile project                     | **`F9`**                            |
| Toggle mm/mil                       | **`Q`**                             |
| Rotate component                    | **`Space`**                         |
| Flip component                      | **`X`** / **`Y`**                   |

> **Note:** Shortcuts can be customised, so if something doesn’t match, check  
> **`DXP → Preferences → PCB Editor → General → Shortcut Keys`**.
